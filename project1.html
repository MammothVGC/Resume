Overview

NileDotCom is a simulated e-commerce application developed as part of UCF coursework. The project focused on building a three-tier distributed web-based application, where customers can browse inventory, place orders, and interact with a dynamic GUI.

Key Features

  • Java-based GUI for product browsing and purchasing.
  • MySQL database integration for inventory management and order storage.
  • Multi-tier architecture (presentation, application, and database layers).
  • Realistic simulation of customer transactions with validation and error handling.

Technologies Used

  • Java (Swing GUI)
  • MySQL with JDBC connector
  • Apache Tomcat (for server deployment, if used)
  • SQL for queries and schema design
